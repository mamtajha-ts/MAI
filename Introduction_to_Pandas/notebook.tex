    \subsubsection{Subsetting Rows Based on
Conditions}\label{subsetting-rows-based-on-conditions}

Often, you want to select rows which satisfy some given conditions. For
e.g., select all the orders where the
\texttt{Sales\ \textgreater{}\ 3000}, or all the orders where
\texttt{2000\ \textless{}\ Sales\ \textless{}\ 3000} and
\texttt{Profit\ \textless{}\ 100}.

Arguably, the best way to do these operations is using
\texttt{df.loc{[}{]}}, since \texttt{df.iloc{[}{]}} would require you to
remember the integer column indices, which is tedious.

Let's see some examples.
